# 🔍 Code Lens — A Powerful Lens into Your Codebase

> Understand, navigate, and act on your codebase — instantly.

Code Lens is a developer productivity tool that acts as an intelligent “lens” over your code. It helps you quickly explore repositories, extract insights, and perform actions like searching, reviewing, and understanding code — all with minimal friction.

---

## 🚀 Why Code Lens?

Modern codebases are massive, fragmented, and hard to navigate. Developers waste hours:

- Searching for relevant files  
- Understanding unfamiliar code  
- Reviewing changes manually  
- Context-switching across tools  

Code Lens solves this by bringing intelligence directly into your workflow.

---

## ✨ Key Features

- Smart Code Search  
- Context-Aware Understanding  
- Instant Insights  
- Actionable Workflows  
- CLI + Agent Integration Ready
